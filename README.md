# bisk-religion-trivia

BISK Religion Trivia — Islamic trivia app with 100 questions (70 standard + 30 hard), spinning wheels, and English, Kurdish, and Arabic support.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages (hosting)

The app is set up to deploy to **GitHub Pages** on every push to `main`.

1. **Enable Pages:** In the repo go to **Settings → Pages**. Under "Build and deployment", set **Source** to **GitHub Actions**.
2. After the workflow runs (or on the next push), the site will be at:  
   **https://bryar1aiq.github.io/bisk-religion-trivia/**
